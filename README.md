## Developing

### Installing

```bash
git clone https://github.com/Roboroads/counting-words.git
cd counting-words
yarn
```

### Running

Check out the [releases](https://github.com/Roboroads/counting-words/releases) for a precompiled app.

For live reloading & development

```bash
yarn dev
```

Build and run a production build

```bash
yarn build
yarn start
```

### Testing

> For the latest test, check the [github actions](https://github.com/Roboroads/counting-words/actions/workflows/test.yml)

Running tests locally

```bash
yarn test
```
